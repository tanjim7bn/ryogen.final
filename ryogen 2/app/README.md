# Ryogen App (dashboard + partner CRM)

This directory is a placeholder for the two authenticated applications:

- **Student Dashboard** — the logged-in student experience (SmartMatch results,
  saved shortlists, application tracker, essays).
- **Partner CRM** — the collaborator workspace (Student CRM, Cohort Analytics,
  Branded Shortlists, Partnership Program).

These are applications, not marketing pages: they need auth, state, and live
data, so they will be built as front-end prototypes (likely React/Next.js to
match the Vercel + framework workflow) and wired to the real backend by the
tech team. Nothing here yet, this is the agreed next build track after the
public marketing site in /site is complete.
